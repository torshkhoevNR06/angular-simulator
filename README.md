# Задачи

[+] Очистить текущий навигационный блок в компоненте header. 
[+] Реализовать массив объектов с навигацией и отобразить через цикл (@for). Он должен содержать такие страницы: [Главная, Пользователи]. 
[] Использовать директивы routerLink, routerLinkActive.
[] Почитать про routerLinkActiveOptions (она пригодится для того, что бы у вас главная страница не светилась по умолчанию).

[] Итоговая цель:  В шапке отображается 2 ссылки: "Главная" и "Пользователи". При клике на ссылку - открывается другая страница (с точки зрения SPA - рисуется другой компонент). Активная ссылка подсвечивается на основе директивы.
